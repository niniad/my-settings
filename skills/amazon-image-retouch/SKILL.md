# Amazon Image Retouch

AI画像編集ツールで既存写真をAmazonトップ商品レベルに加工するためのプロンプトテンプレート集。

## ワークフロー

1. ユーザーが加工したい写真を共有（またはパスを指定）
2. 写真の用途を確認 → カテゴリ選択ガイドで該当カテゴリを特定
3. [references/prompts.md](references/prompts.md) から該当カテゴリのベースプロンプトを取得
4. `[placeholders]` に商品固有の情報を記入してユーザーに提示
5. ユーザーがAIツール（Nano Banana Pro / ChatGPT / Adobe Firefly / Photoroom等）でプロンプトを実行

## プロンプト言語ルール

- プロンプトは**すべて英語**で記述する
- 日本語テキストをプロンプト内に混在させない（AIツールの安定性低下防止）
- 画像内に日本語テキストが必要な場合は後工程（Affinity Designer等）で追加
- `[placeholders]` を商品情報で適切に置換する

## 品質ベンチマーク

Amazon BSR Top 10（BOTANIST, Ancker等）の画像を基準とする。

| 要素         | 基準                                                                 |
| ------------ | -------------------------------------------------------------------- |
| 背景         | 純白（#FFFFFF）シームレス。影はコントロールされたもののみ            |
| ライティング | 多方向から均一なソフトライト + 1箇所のスペキュラーハイライトで立体感 |
| 表面         | 傷ゼロ・ホコリゼロ・指紋ゼロ。CGレンダー級の美しさ                   |
| シャープネス | タックシャープ。素材の質感・ラベルテキストまで鮮明                   |
| 色           | 実物に忠実かつ鮮やか（+5-10%）。色かぶりなし                         |
| エッジ       | クリーンカット。フリンジ・白縁・色にじみなし                         |

## カテゴリ一覧

| #   | カテゴリ         | 用途                        | 背景                   |
| --- | ---------------- | --------------------------- | ---------------------- |
| 1   | メイン画像       | 画像1（検索結果サムネイル） | 純白必須（Amazon規約） |
| 2   | 物撮り（白背景） | サブ画像・A+ の商品写真     | 白背景推奨（制約なし） |
| 3   | ライフスタイル   | 使用シーン・生活空間        | 自然な環境             |
| 4   | 人物 + 商品      | 権威付け・推薦・体験        | 白背景 or 環境         |
| 5   | 比較・問題提起   | Before/After・課題提示      | 白〜ライトグレー       |

## カテゴリ選択ガイド

```
写真の用途は？
├─ メイン画像（画像1）     → カテゴリ 1
└─ サブ画像 / A+
    ├─ 白背景の商品写真    → カテゴリ 2
    ├─ 使用シーン・生活空間 → カテゴリ 3
    ├─ 人物が写っている     → カテゴリ 4
    ├─ 問題点・比較を見せる → カテゴリ 5
    └─ よく分からない       → クイックレタッチ
```

各カテゴリのプロンプトテンプレートは [references/prompts.md](references/prompts.md) を参照。
