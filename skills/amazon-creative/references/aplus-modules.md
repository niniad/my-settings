# A+ コンテンツ モジュールリファレンス

## モジュール一覧

| # | モジュール名 | 画像サイズ | テキスト | 最適な訴求 |
|---|------------|-----------|---------|-----------|
| 1 | スタンダード画像とテキスト | 970×600px | 見出し＋本文 | ブランドストーリー、主要機能の大きなビジュアル |
| 2 | 4つの画像とテキスト | 各220×220px（4枚） | 各画像下にタイトル＋説明 | 複数機能の並列説明（モバイルは2×2） |
| 3 | 比較表 | 各150×300px（最大5商品） | 比較項目マトリクス | バリエーション比較、競合比較（※直接名指しNG） |
| 4 | テキスト付き画像（左） | 300×300px | 右側に見出し＋本文 | 機能の詳細解説 |
| 5 | テキスト付き画像（右） | 300×300px | 左側に見出し＋本文 | 4の反転版。交互配置でリズムを作る |
| 6 | 画像とテキストのオーバーレイ | 970×600px | 画像上にオーバーレイ | インパクト重視のビジュアル訴求 |
| 7 | 3つの画像とテキスト | 各300×300px（3枚） | 各画像下にタイトル＋説明 | 3ステップの使用方法、3つの特徴 |
| 8 | スタンダードテキスト | なし | 見出し＋本文のみ | 詳細説明、注意事項、FAQ |

## モジュール選択フロー

```
訴求内容を確認
├─ 複数商品/バリエーション比較 → 比較表
├─ 複数ポイント並列
│   ├─ 4つ → 4つの画像とテキスト
│   └─ 3つ → 3つの画像とテキスト
├─ 大きなビジュアル重視 → 画像とテキスト or オーバーレイ
├─ 1機能の詳細解説 → テキスト付き画像(左/右)
├─ テキスト中心の説明 → スタンダードテキスト
└─ ブランドストーリー → 画像とテキスト
```

## テキスト文字数制限（目安）

- 見出し: 全角50文字程度
- 本文: 全角300文字程度

※Seller Centralの入稿画面で最新の制限を確認すること。

## 注意事項

- 画像内テキストはAmazon検索にインデックスされない → テキストフィールドにも同等内容を記載
- モバイルで画像は縮小表示 → 画像内テキストのサイズに注意
- 全モジュールを使う必要はない。3-5モジュール程度が適切
