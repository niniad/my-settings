# Amazon A+ コンテンツ モジュールリファレンス

## 概要

A+コンテンツ（旧EBC: Enhanced Brand Content）は、ブランド登録済みセラーが使用できる商品説明の強化機能。テキストだけの商品説明を、画像とテキストを組み合わせたリッチなレイアウトに置き換えられる。

## 利用可能なモジュール一覧

### 1. スタンダード画像とテキスト（Standard Image & Text）

**画像サイズ**: 970×600px
**テキスト**: 見出し＋本文（HTMLタグ一部利用可）

**最適な訴求**:
- ブランドストーリー
- 主要機能の写真付き解説
- 大きなビジュアルで魅せたい場面

**レイアウト**: 画像が幅いっぱい、その下にテキスト

### 2. スタンダード4つの画像とテキスト（Standard Four Images & Text）

**画像サイズ**: 各220×220px（4枚）
**テキスト**: 各画像の下にタイトル＋説明

**最適な訴求**:
- 複数機能の並列説明
- 4つのポイント・特徴紹介
- アクセサリ・付属品一覧

**レイアウト**: 4列横並び（モバイルは2×2グリッド）

### 3. スタンダード比較表（Standard Comparison Table）

**画像サイズ**: 各150×300px（最大5商品）
**テキスト**: 比較項目×商品のマトリクス

**最適な訴求**:
- 自社商品のバリエーション比較
- 競合との仕様比較
- スペック一覧

**レイアウト**: 表形式。各列に商品画像、行に比較項目
**注意**: 競合商品名を直接記載するのはAmazonポリシー違反リスクあり。「従来品」「一般的な製品」等の表現を使う。

### 4. スタンダードテキスト付き画像（左）

**画像サイズ**: 300×300px
**テキスト**: 画像の右側に見出し＋本文

**最適な訴求**:
- 機能の詳細解説（1つずつ丁寧に）
- 素材の特徴説明

**レイアウト**: 左に画像、右にテキスト

### 5. スタンダードテキスト付き画像（右）

**画像サイズ**: 300×300px
**テキスト**: 画像の左側に見出し＋本文

4の左右反転版。複数モジュールを交互に配置して視覚的リズムを作る。

### 6. 画像とテキストのオーバーレイ

**画像サイズ**: 970×600px
**テキスト**: 画像上にオーバーレイ表示

**最適な訴求**:
- インパクトのあるビジュアル訴求
- ブランドイメージ
- 商品の世界観表現

**注意**: テキストの可読性を確保するため、画像のコントラストに注意

### 7. スタンダード3つの画像とテキスト

**画像サイズ**: 各300×300px（3枚）
**テキスト**: 各画像の下にタイトル＋説明

**最適な訴求**:
- 3ステップの使用方法
- 3つの主要特徴
- 3つのカラーバリエーション

### 8. スタンダードテキスト

**画像**: なし
**テキスト**: 見出し＋本文のみ

**最適な訴求**:
- 詳細な商品説明・ストーリー
- 注意事項・免責事項
- FAQ

## モジュール選択のフローチャート

```
訴求内容を確認
├─ 複数商品/バリエーション比較 → 比較表
├─ 複数ポイント並列
│   ├─ 4つ → 4つの画像とテキスト
│   └─ 3つ → 3つの画像とテキスト
├─ 大きなビジュアル重視 → 画像とテキスト or オーバーレイ
├─ 1機能の詳細解説 → テキスト付き画像(左/右)
├─ テキスト中心の説明 → スタンダードテキスト
└─ ブランドストーリー → 画像とテキスト
```

## A+コンテンツの配置順序

サブ画像のストーリーを引き継ぎ、以下の流れで構成する:

```
1. キャッチ（画像とテキスト or オーバーレイ）
   → サブ画像で興味を持ったユーザーを引き込む
2. 詳細解説（テキスト付き画像 × 2-3モジュール）
   → サブ画像では伝えきれなかった機能の深掘り
3. 比較（比較表）
   → 競合/バリエーション比較で優位性を明示
4. 使い方（3つの画像とテキスト）
   → 使用方法をステップで解説
5. ブランドストーリー（画像とテキスト）
   → ブランドへの信頼・親近感
```

全モジュールを使う必要はない。商品に合わせて3-5モジュール程度が適切。

## 制作時の注意点

### テキストの文字数制限

各モジュールのテキストフィールドには文字数制限がある。Seller Centralの入稿画面で確認すること。おおよその目安:

- 見出し: 全角50文字程度
- 本文: 全角300文字程度

### 画像内テキストの注意

A+の画像内にテキストを入れる場合:
- モバイル表示で読めるサイズか確認（画像は縮小表示される）
- テキストフィールドにも同等の内容を記載（SEO・アクセシビリティ）
- Amazon検索アルゴリズムは画像内テキストをインデックスしない

### 禁止事項

- 主観的な品質主張（「業界最高」「最安値」等）の裏付けのない使用
- 保証・返品ポリシーへの言及（Amazonのポリシーに抵触）
- 外部サイトへの誘導
- 競合商品名の直接記載
- 時限的なプロモーション情報
